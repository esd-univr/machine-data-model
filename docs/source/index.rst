Glacier Machine Data Model documentation
========================================

**Glacier Machine Data Model** is a Python library for creating and managing machine data models.
It is part of the Glacier project, which aims to provide a comprehensive set of tools for the development of digital twins of manufacturing systems.
The library is designed to provide a simple and unified interface for creating, managing, and querying machine data models.
Machine data models can be connected to real or simulated machines, sensors, and actuators.

.. note::
   This project is under active development.
